Spotify Insights Dashboard

Overview:
A professional Streamlit-based web application designed to analyze and visualize the Spotify-2000 dataset. This tool provides deep insights into musical trends, emotional attributes, and popularity metrics through an interactive, Spotify-themed interface.

Key Features:

Interactive Filtering: Allows users to dynamically filter data by musical genre using sidebar controls.

Sentiment Analysis (Valence): Visualizes the average "happiness" level of tracks across various genres using interactive bar charts.

Rhythm & Tempo Tracking: Analyzes song distribution based on Beats Per Minute (BPM) to identify tempo trends.

Mood vs. Popularity Correlation: Explores the relationship between a track's valence (mood) and its popularity score using detailed scatter plots.

Custom Dark UI: Features a sleek, Spotify-inspired dark theme built with custom CSS for an immersive experience.

Tech Stack:

Framework: Streamlit

Data Analysis: Pandas

Visualization: Plotly and Matplotlib
